\# ⚡ XLFaults



Ferramenta computacional para análise de curto-circuito em sistemas elétricos de potência, com entrada de dados e processamento totalmente integrados ao Excel.



\---



\## 📌 Introdução e Motivações



Ao analisar as ferramentas disponíveis para estudantes e profissionais de Engenharia Elétrica, é comum encontrar soluções pagas, versões acadêmicas com limitações no número de barramentos ou até mesmo softwares com instabilidades e bugs.



Além disso, a organização de sistemas elétricos de potência é naturalmente mais eficiente quando estruturada em tabelas — algo que o Excel faz de forma simples, intuitiva e poderosa.



Pensando nessas limitações, desenvolvi o \*\*XLFaults\*\*: uma ferramenta de análise de curto-circuito sem restrições de barramentos, integrada diretamente ao Excel e voltada para praticidade, flexibilidade e autonomia do usuário.



A proposta é simples:



> Você precisa apenas do código e de uma planilha Excel bem estruturada.  

> A partir disso, toda a modelagem, processamento e análise acontecem de forma automatizada — é só preparar os dados e deixar a mágica acontecer.



\---



\## ⚙️ Funcionalidades



\- Entrada de dados estruturada via Excel

\- Modelagem automática do sistema elétrico

\- Construção das matrizes do sistema (como Ybarra)

\- Cálculo de curto-circuito com:

&#x20; - Falta trifásica

&#x20; - Falta monofásica

&#x20; - Falta bifásica

&#x20; - Falta bifásica-terra

&#x20; - Consideração de impedância de falta

\- Disponibilização das matrizes para o usuário

\- Geração automática de diagramas unifilares



\---



\## 📊 Exemplos



O fluxo de uso da ferramenta segue a lógica:



1\. O usuário preenche os dados do sistema em uma planilha Excel

2\. O código realiza automaticamente:

&#x20;  - Leitura e organização dos dados

&#x20;  - Montagem do modelo elétrico

&#x20;  - Cálculo das faltas

3\. São gerados como saída:

&#x20;  - Correntes de curto-circuito

&#x20;  - Tensões nos barramentos

&#x20;  - Matrizes do sistema

&#x20;  - Diagrama unifilar do sistema modelado
